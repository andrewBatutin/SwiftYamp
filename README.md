## SwiftYamp

Swift implementation of the [Yamp](https://github.com/yyyar/yamp) protocol for iOS

Supports WebSocket transport

Install:

Carthage

`Podfile`:

  ```ogdl
  platform :ios, '10.0'

  # You need to set target when you use CocoaPods 1.0.0 or later.
  target 'SampleTarget' do
    use_frameworks!
    pod 'SwiftYamp'
  end
  ```
